This Stack Assignment solution can be found in StackAssignment.sln file.
The compilation is done by using the Visual Studio 2022's C++ compiler in Windows 11.

Tests use Google Tests. For the test framework 'cmake .' command needs to be run.
See CMakeLists.txt for details what it does. 

The actual tests solution is generated by the cmake command and can be found in 
GoogleTestsForStackAssignment.sln solution file. To run the tests compile the 
solution and set StackUnitTests project as start up project in Visual studio.

Doxygen HTML file for stack documentation can be found in html/index.html file
